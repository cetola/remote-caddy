# remote-caddy

A simple, small, remote control caddy that holds an Apple TV remote as well as a remote for the MULTICOMP PRO IR Remote Volume Control (line level). Manufacturer Part No 50-8394 Newark Part No. 79X4195.

## Repository Structure

- `CAD/`  
  FreeCAD source files (`.FCStd`). These are the authoritative, parametric design sources.

- `STLs/`  
  Exported STL files intended for 3D printing.

- `docs/`  
  Documentation and reference images.
